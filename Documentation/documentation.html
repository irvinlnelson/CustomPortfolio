4. CSS class (margin, padding) - Top

You can use this CSS class:

Margin: m-t-*, m-b-*, m-l-*, m-r-* (Instead * you can put this values: 10, 20, 30, 40, 50)
Padding: p-t-*, p-b-*, p-l-*, p-r-* (Instead * you can put this values: 10, 20, 30, 40, 50)

Example:
<p class="m-l-10"> = <p style="margin-left: 10px;">
<p class="p-t-50"> = <p style="padding-top: 50px;">
<p class="m-b-30"> = <p style="margin-bottom: 30px;">
<p class="p-r-40"> = <p style="padding-right: 40px;">

7. How to switch color palette theme? - Top

You can switch color palette by changing this line <link rel="stylesheet" href="css/themes/theme-orange.css"> in index.html on line 11.

Available values: "theme-blue.css", "theme-brown.css", "theme-grey.css", "theme-orange.css", "theme-pink.css", "theme-purple.css".
